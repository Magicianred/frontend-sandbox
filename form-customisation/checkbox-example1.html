Custom checkbox

Example 1

Hides the browser native checkbox and displays a custom checkbox. Makes use of the general sibling ~ and the CSS3 :checked selectors to toggle the checkbox without JavaScript.

Support

Supported in all CSS3-capable browsers. Displays a standard checkbox component in non-supporting browsers.

  • Chrome
  • Chrome Android
  • Safari
  • Mobile Safari
  • Opera
  • Firefox
  • IE9/10/11

Accessibility

It provides the same accessibility as a standard component.
